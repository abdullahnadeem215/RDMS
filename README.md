# 👋 Welcome

The **Rural Development Management System (RDMS)** is a centralized web-based application developed to streamline and organize rural welfare and development activities.

Traditional NGO record systems often rely on scattered spreadsheets and manual records. RDMS solves this problem through a centralized management platform with secure access and structured database architecture.

The system was designed as a practical implementation of **Database Management System concepts** with real-world organizational workflows.

# 🚀 Installation Guide

### Step 1 — Clone Repository

```bash
git clone https://github.com/abdullahnadeem215/RDMS.git
```

---

### Step 2 — Move Project

Copy project into:

```text
C:\xampp\htdocs\
```

Result:

```text
C:\xampp\htdocs\RDMS
```

---

### Step 3 — Start XAMPP

Open XAMPP and run:

✅ Apache

✅ MySQL

---

### Step 4 — Database Setup

Open:

```text
http://localhost/phpmyadmin
```

Create database:

```text
rdms_usf
```

Import:

```text
Backend/rdms_database.sql
```

---

### Step 5 — Verify Configurations

Default:

```env
Host=localhost
User=root
Password=
Database=rdms_usf
Port=3307
```

Update:

```php
Backend/config.php
```

if needed.

---

### Step 6 — Run Application

```text
http://localhost/RDMS/
```
